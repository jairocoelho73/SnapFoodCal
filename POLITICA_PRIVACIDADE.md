# Politica de Privacidade - SnapFoodCal

**Ultima atualizacao:** 05 de Dezembro de 2024

O SnapFoodCal ("nos", "nosso" ou "aplicativo") respeita sua privacidade e esta comprometido em proteger seus dados pessoais. Esta politica de privacidade explica como coletamos, usamos e protegemos suas informacoes quando voce usa nosso aplicativo.

## 1. Informacoes que Coletamos

### 1.1 Dados de Saude e Fitness
- Peso, altura e idade (inseridos voluntariamente)
- Metas de calorias e macronutrientes
- Historico de refeicoes e consumo de agua
- Registros de atividades fisicas
- Dados de jejum intermitente

### 1.2 Fotos de Alimentos
- Fotos tiradas ou selecionadas para analise nutricional
- Codigos de barras escaneados para identificacao de produtos

### 1.3 Dados de Autenticacao
- PIN de acesso (armazenado de forma criptografada)
- Configuracoes de biometria (Face ID / Touch ID)

## 2. Como Usamos Suas Informacoes

### 2.1 Analise Nutricional
As fotos de alimentos sao enviadas para a API do Google Gemini exclusivamente para analise nutricional. O Google processa as imagens em tempo real e nao as armazena apos o processamento.

### 2.2 Funcionamento do App
Seus dados de saude sao usados para:
- Calcular seu IMC e TDEE
- Acompanhar seu progresso diario
- Fornecer sugestoes personalizadas
- Gerenciar planos alimentares e jejum

### 2.3 O Que NAO Fazemos
- NAO vendemos seus dados para terceiros
- NAO usamos seus dados para publicidade direcionada
- NAO compartilhamos informacoes pessoais sem seu consentimento

## 3. Armazenamento de Dados

### 3.1 Armazenamento Local
Todos os seus dados pessoais sao armazenados **exclusivamente no seu dispositivo**. Isso inclui:
- Historico de refeicoes
- Dados de perfil (peso, altura, metas)
- Configuracoes do aplicativo
- Registros de jejum e atividades

### 3.2 Nenhum Servidor Externo
O SnapFoodCal nao possui servidores proprios para armazenamento de dados de usuarios. Seus dados permanecem sob seu controle total.

### 3.3 Processamento na Nuvem
O unico dado enviado externamente sao as fotos de alimentos, processadas pela API do Google Gemini para analise. Essas imagens:
- Sao transmitidas de forma criptografada (HTTPS)
- Sao processadas em tempo real
- NAO sao armazenadas pelo Google apos o processamento

## 4. Seguranca

Implementamos as seguintes medidas de seguranca:

- **Criptografia:** O PIN de acesso e armazenado de forma criptografada usando Expo SecureStore
- **Biometria:** Suporte a Face ID e Touch ID para acesso rapido e seguro
- **Bloqueio Automatico:** O app bloqueia automaticamente quando voce sai dele
- **Transmissao Segura:** Todas as comunicacoes usam protocolo HTTPS

## 5. Seus Direitos

De acordo com a LGPD (Lei Geral de Protecao de Dados) e GDPR, voce tem direito a:

### 5.1 Acesso e Portabilidade
Voce pode exportar todos os seus dados a qualquer momento atraves de:
**Perfil > Sobre > Exportar Meus Dados**

### 5.2 Exclusao
Voce pode excluir permanentemente todos os seus dados atraves de:
**Perfil > Sobre > Excluir Meus Dados**

### 5.3 Correcao
Voce pode editar suas informacoes pessoais a qualquer momento nas configuracoes do aplicativo.

## 6. Uso por Menores

O SnapFoodCal e classificado para todas as idades. No entanto, recomendamos que menores de 13 anos utilizem o aplicativo sob supervisao de um adulto.

## 7. Alteracoes nesta Politica

Podemos atualizar esta politica de privacidade periodicamente. Notificaremos sobre mudancas significativas atraves do aplicativo. A data de "ultima atualizacao" no topo indica quando a politica foi revisada pela ultima vez.

## 8. Servicos de Terceiros

### 8.1 Google Gemini API
Usamos a API do Google Gemini para analise de imagens de alimentos. O uso esta sujeito a:
- [Politica de Privacidade do Google](https://policies.google.com/privacy)
- [Termos de Servico do Google AI](https://ai.google.dev/terms)

### 8.2 Open Food Facts
Para produtos com codigo de barras, consultamos o banco de dados Open Food Facts (dados publicos e abertos).
